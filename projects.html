PocketASD

PocketASD aims to provide parents of children with Autism Spectrum Disorder (ASD) with helpful resources. Patients can view videos, take a screener, and request appointments in a mobile app. Moderators use an online portal to view requests, patient information, and more.

Interactive Map

Interactive map project that uses Google Maps API for an interactive map.
